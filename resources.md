# Course Resources

## [Github Tutorial: Hello World](https://docs.github.com/en/get-started/start-your-journey/hello-world)
This tutorial from Github goes over the basics of creating and using repositories, including commits, pull requests, and branches. 

## [Agile vs Waterfall](https://www.ibm.com/think/topics/agile-vs-waterfall)
This IBM article covers the difference between the Agile and Waterfall methods, highlighting the strengths of each. 

## [Agile vs Scrum](https://graduate.northeastern.edu/knowledge-hub/agile-vs-scrum/)
This article from Northeastern University lists some of the differences between the relatively similar Agile and Scrum methods, helping further refine what development method might be best for a particular project. 

## [Project Libre](https://www.projectlibre.com/)
Project Libre is a tool for creating diagrams, including Gantt charts that are useful for project management.
